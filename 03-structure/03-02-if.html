La condition if est la plus utilisée des conditions

If

1- if...

La condition if est l'une des plus utilisée et est également la plus simple à appréhender puisqu'elle va nous permettre d'exécuter un bloc de code si Et seulment si le résultat vaut true.

Attention !

tout test d'une condition va être évalué dans contexte booléen , danc à true à l'execption des valeurs suivantes qui vont être évaluées à false

  • la valeur 0(Zéro)
  • le booléen false
  • une chaîne de carctére vide
  • la valeur null
  • la valeur undefined
  • la valeur NAN(NOT a Number)

3 - Invzeser la valeur logique d'un test

Le code placé dans le if n'est exécuté que si le résultat de la comparaison est true.

Dans certaines situations,on préfère vérifier si la comparaison est false : on peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false.
