JavaScript - 02/02 AFFICHAGE

Comment afficher des variables

1- sept types de données en js

  1. String ou chaîne de caractères
  2. Number ou nombre
  3. Boolean ou booléen
  4. Null ou "vide"
  5. Undefined ou indéfini
  6. symbol ou symbole
  7. Object ou objet

pour connaître les types de données on utilise la fonction typeof

1-String

ce contenu sera remplacé

a prpos des guillemets "....", des quotes et sur le caractère d'échappement '....' ou antislash \

En JavaScript, on va pouvoir utiliser indifféremment des guillemets doubles ou des quotes pour entourer une chaîne de caractères et ces deux méthodes vont être strictement équivalentes à la différence d’autres langages comme le PHP par exemple.
Faites attention cependant à un point : si votre chaîne contient un caractère qui est le même que le délimiteur de chaîne choisi, il faudra neutraliser ce caractère en l’échappant au moyen d’un antislash ou changer de délimiteur.

2-number

Les variables en JavaScript vont également pouvoir stocker des nombres. Des nombres entiers positifs ou négatifs ou à virgule (décimaux)

3- Bolean

ce type de données sera utile pour vérifier si un test est vrai ou faux

un booléen, en algèbre est un type qui ne contient que deux valeaurs: les valeurs true (vrai) et false (faux).

4 - NULL

NULL est un type un peu particulier : null absence de valeur connue ou "vide"

Undefined

Undefined : variable non définie

si on déclare une variable sans lui attribuer une valeur, alors son type sera Undefined

6- Symbol et 7- Object

Les objets sont des structures complexes qui vont pouvoir stocker plusieurs valeurs en même temps et que nous étudierons plus tard car il s’agit là d’un sujet complexe à appréhender pour un débutant.

Si on déclare une variable et qu’on lui passe null, alors son type sera Object.
