les opérateurs logiques

ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat du résultat d'un test.

Opérateurs logiques
Nom Symbole Description
AND (et) && renvoie true si toutes les comparaisons sont vraies ou false sinon.
OR (ou) || renvoie true si au moins l'une des comparaisons est vraie ou false sinon
NO (non) ! renvoie true si une comparaison est évalué à false.
